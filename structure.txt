【目录结构】
demo/
├── index.html          # 主入口（精简后的HTML）
├── styles.css          # 样式文件
├── ssd-gc-simulator.html # 原文件（保留备份）
└── js/
    ├── config.js       # 配置管理模块
    ├── state.js        # 状态管理模块
    ├── utils.js        # 工具函数模块
    ├── writeStrategy.js    # 写入策略模块
    ├── gcStrategy.js       # GC策略模块
    ├── renderer.js     # 渲染模块
    ├── app.js          # 前端交互模块
    └── main.js         # 入口组装模块

【三层架构拓扑】
┌─────────────────────────────────────────────────────────┐
│                    前端显示层级                          │
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────────┐ │

│  │  (CSS样式)  │  │ (SSD可视化)  │  │ (按钮/弹窗/日志)│ │
│  └─────────────┘  └─────────────┘  └─────────────────┘ │
├─────────────────────────────────────────────────────────┤
│                   SSD模拟器层级                          │
│  ┌─────────────────┐  ┌────────────────────────────────┐│
│  │   写入策略模块   │  │          GC策略模块            ││
│  │  - 顺序写入     │  │  - Victim选择策略               ││
│  │  - 随机写入     │  │  - 执行流程（读→擦→写）         ││
│  │  - PSB分配      │  │  - 步进式GC演示                 ││
│  └─────────────────┘  └────────────────────────────────┘│
├─────────────────────────────────────────────────────────┤
│                   基础设施层级                           │
│  ┌───────────┐  ┌───────────┐  ┌─────────────────────┐ │
│  │ 配置管理   │  │ 状态管理   │  │   工具函数          │ │
│  │ (CONFIG)  │  │ (ssdState)│  │ (日志/历史/工具)    │ │
│  └───────────┘  └───────────┘  └─────────────────────┘ │
└─────────────────────────────────────────────────────────┘

【详细模块说明】
模块		职责				对外接口
config.js		配置参数、默认参数恢复		CONFIG 对象
state.js		SSD状态、映射表、历史快照		initSSD(), saveState(), undo(), redo()
utils.js		日志、状态更新、高亮		addLog(), updateStatus(), highlightPages()
writeStrategy.js	顺序/随机写入、PSB选择		sequentialWrite(), randomWrite()
gcStrategy.js	Victim选择、GC执行、步进演示	triggerGC(), performGC(), selectVictimPsb()
renderer.js	PSB渲染、映射表渲染		renderSSD(), updateMappingTable()
app.js		按钮事件、弹窗控制			initApp()


【架构层次】
----------------------------------------------------------------------------------------------------------
层级			模块			职责

前端显示层		renderer.js		SSD可视化、映射表渲染
			styles.css			所有样式定义

SSD模拟器层		writeStrategy.js		顺序写入、随机写入、PSB分配策略
			gcStrategy.js		Victim选择、GC执行、步进式演示

基础设施层		config.js			配置参数管理
			state.js			SSD状态、映射表、历史记录
			utils.js			日志、状态更新、高亮等辅助功能

应用入口层		app.js			初始化逻辑
			main.js			模块组装、全局函数暴露
----------------------------------------------------------------------------------------------------------
